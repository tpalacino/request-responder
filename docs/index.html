Request Responder

⚡ Fast, Flexible, and Private Request Redirection for Web Developers

Version: 1.1.0 Released: 5/5/2026

What is Request Responder?

Request Responder is a browser extension that allows web developers to easily manage rules to redirect network requests from one endpoint to another. It provides a powerful and intuitive interface for creating and testing declarativeNetRequest rules.

Key Features

  • Intuitive Rule Management: Create, edit, and organize your request redirection rules with ease.
  • Real-Time Testing: Test your rules in real-time to see how they affect network requests.
  • Privacy-Focused: Request Responder is designed with a strict no-data collection policy, ensuring your privacy is always protected. Privacy Policy
  • Open Source: The source code is available on GitHub, allowing for transparency and community contributions.
